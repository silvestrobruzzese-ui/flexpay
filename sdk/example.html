📋 Codice di Integrazione

Aggiungi questo codice al tuo sito:

// 1. Includi lo script <script src="https://cdn.flexpay.com/sdk.js"></script> // 2. Inizializza con la tua public key <script> FlexPay.init('pk_live_your_key_here'); // 3. Crea il bottone di pagamento FlexPay.createButton('#flexpay-button', { amount: 99.00, currency: 'EUR', description: 'Prodotto Premium', successUrl: 'https://tuosito.com/success', cancelUrl: 'https://tuosito.com/cancel', onSuccess: function(data) { console.log('Pagamento completato!', data); }, onCancel: function(data) { console.log('Pagamento annullato'); } }); </script>

🔧 Integrazione Backend

Per sicurezza, crea le sessioni dal tuo server:

// Node.js - Crea sessione di checkout const response = await fetch('https://api.flexpay.com/api/checkout/sessions', { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-API-Key': 'sk_live_your_secret_key' }, body: JSON.stringify({ amount: 99.00, currency: 'EUR', description: 'Ordine #12345', customerEmail: 'cliente@email.com', successUrl: 'https://tuosito.com/success', cancelUrl: 'https://tuosito.com/cancel', metadata: { orderId: '12345' } }) }); const session = await response.json(); // Redirect utente a session.url
